# Tech Burnout Predictor

## โปรเจกต์นี้ทำอะไร

ใช้ข้อมูลจำลองของคนทำงานสายเทค ให้โมเดล Machine Learning ช่วยจัดกลุ่มระดับความเสี่ยง **ภาวะหมดไฟ (Burnout)** เป็น **Low / Moderate / High**

มีสองส่วนหลัก:

1. **โน้ตบุ๊ก** `burnout_.ipynb` — อ่านข้อมูล ฝึกโมเดล แล้วบันทึก `burnout_model.pkl` (เมื่อต้องการเทรนใหม่)
2. **แอป Streamlit** `app.py` — กรอกแบบประเมิน ดูผลทันที กราฟความน่าจะเป็น และความสำคัญของฟีเจอร์

ผลจากแอปใช้เพื่อ **การเรียนรู้ในรายวิชา** ไม่ใช่การวินิจฉัยโรค และไม่ใช่คำแนะนำทางการแพทย์

---

## สิ่งที่ต้องมี

- **Python 3.10 ขึ้นไป**
- โคลนโปรเจกต์หรือดาวน์โหลดโฟลเดอร์นี้ — ใน repo มี **`burnout_model.pkl`** และ **`tech_mental_health_burnout.csv`** อยู่แล้ว (ใช้รันแอปและฝึกในโน้ตบุ๊กได้ทันที)

---

## ติดตั้งแพ็กเกจ (ครั้งแรก)

```text
cd โฟลเดอร์โปรเจกต์นี้
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

(บน macOS / Linux ใช้ `source .venv/bin/activate`)

ถ้าจะเปิดโน้ตบุ๊ก แนะนำติดตั้ง Jupyter เพิ่ม: `pip install jupyter`

---

## รันแอป Streamlit

```text
streamlit run app.py
```

เปิดเบราว์เซอร์ที่ `http://localhost:8501` กรอกฟอร์มแล้วกดวิเคราะห์

---

## เทรนโมเดลใหม่ (ถ้าต้องการ)

1. เปิด **`burnout_.ipynb`** ด้วย Jupyter หรือ VS Code / Cursor  
2. เลือก Python environment เดียวกับที่ติดตั้งแพ็กเกจ  
3. รันเซลล์ตามลำดับจนถึงเซลล์ที่บันทึก **`burnout_model.pkl`**  
4. วางไฟล์ `burnout_model.pkl` ไว้ **โฟลเดอร์เดียวกับ `app.py`** แล้วรันแอปใหม่  

ถ้าไฟล์โมเดลใหญ่เกินกว่าจะ push ขึ้น GitHub ได้ ให้ใส่ชื่อไฟล์ใน `.gitignore` แล้วสร้างในเครื่องหลัง clone

---

## ไฟล์หลัก

| ไฟล์ | บทบาท |
|------|--------|
| `app.py` | แอป Streamlit |
