# Tech Burnout Predictor

## โปรเจกต์นี้ทำอะไร

โปรเจกต์นี้ใช้ข้อมูลจำลองของคนทำงานสายเทค แล้วให้โปรแกรมเรียนรู้ (Machine Learning) ช่วยประเมินว่าโอกาสที่จะอยู่ในกลุ่ม **Low / Moderate / High** ตามระดับความเสี่ยง **ภาวะหมดไฟ (Burnout)** เป็นอย่างไร

มีสองส่วนหลัก:

1. **โน้ตบุ๊ก** — อ่านข้อมูล ฝึกโมเดล แล้วบันทึกเป็นไฟล์ `burnout_model.pkl`
2. **แอปเว็บ (Streamlit)** — ให้กรอกแบบคำถามแล้วดูผลทันที

ผลจากแอปเป็นเพียง **ตัวอย่างการเรียนรู้ในรายวิชา** ไม่ใช่การวินิจฉัยโรค และไม่ใช่คำแนะนำทางการแพทย์

---

## สิ่งที่ต้องมีก่อนเริ่ม

- ติดตั้ง **Python 3.10 ขึ้นไป**
- ไฟล์ข้อมูล **`tech_mental_health_burnout.csv`** อยู่ในโฟลเดอร์โปรเจกต์เดียวกับโค้ด

---

## ติดตั้งแพ็กเกจ (ครั้งแรก)

เปิดเทอร์มินัล แล้วเข้าโฟลเดอร์โปรเจกต์นี้

แนะนำให้สร้าง virtual environment แล้วเปิดใช้งาน จากนั้นรัน:

```text
pip install -r requirements.txt
```

บน Windows ถ้าใช้ venv ชื่อ `.venv` การเปิดใช้ใน PowerShell มักเป็น:

```text
.\.venv\Scripts\Activate.ps1
```

---

## ขั้นตอนที่ 1: สร้างไฟล์โมเดล `burnout_model.pkl`

แอปจะอ่านโมเดลจากไฟล์ชื่อ **`burnout_model.pkl`** ที่อยู่ **โฟลเดอร์เดียวกับ `app.py`**

วิธีที่ตรงกับเนื้อหาในรายวิชา:

1. เปิด **`Untitled0.ipynb`** ด้วย Jupyter หรือ VS Code / Cursor
2. เลือก Python environment เดียวกับที่ติดตั้งแพ็กเกจ
3. รันเซลล์ **ตามลำดับจากบนจนถึงเซลล์ที่สั่ง `joblib.dump`** เพื่อบันทึก `burnout_model.pkl`
4. ตรวจว่าในโฟลเดอร์โปรเจกต์มีไฟล์ **`burnout_model.pkl`** แล้ว

**ทางลัด:** ถ้าไม่ใช้โน้ตบุ๊ก สามารถรัน `python train_save_model.py` ได้ (สคริปต์ทำหน้าที่คล้ายการฝึกในโน้ตบุ๊ก แล้วบันทึกไฟล์โมเดลให้)

หมายเหตุ: ไฟล์โมเดลมักไม่ใส่ใน Git เพราะขนาดใหญ่ ต้องสร้างในเครื่องเองหลัง clone โปรเจกต์

---

## ขั้นตอนที่ 2: เปิดแอป Streamlit

เมื่อมี `burnout_model.pkl` แล้ว รัน:

```text
streamlit run app.py
```

โปรแกรมจะเปิดหน้าเว็บในเบราว์เซอร์ (ปกติที่ `http://localhost:8501`) ให้กรอกฟอร์มแล้วกดวิเคราะห์ได้

---

## ไฟล์หลักในโปรเจกต์

| ไฟล์ | คำอธิบายสั้นๆ |
|------|----------------|
| `tech_mental_health_burnout.csv` | ข้อมูลใช้ฝึกโมเดล |
| `Untitled0.ipynb` | วิเคราะห์ข้อมูล ฝึกโมเดล บันทึก `burnout_model.pkl` |
| `train_save_model.py` | ทางเลือกสร้าง `burnout_model.pkl` จากเทอร์มินัล |
| `app.py` | หน้าเว็บ Streamlit |
| `requirements.txt` | รายการแพ็กเกจ Python |
| `.streamlit/config.toml` | ตั้งค่าธีมของ Streamlit (ถ้ามี) |
